# 🧬 AI Digital Health Twin: Personalized Health Risk Prediction System

## 🚀 Overview
AI Digital Health Twin is an intelligent system that simulates a user's health profile to predict risk levels and provide personalized recommendations using Machine Learning, RAG, and Multi-Agent AI.

This project demonstrates how AI can be used for preventive healthcare, real-time monitoring, and intelligent decision-making.

---

## 💡 Key Features

- 📈 Health risk prediction using Machine Learning
- 📊 Risk classification (High / Medium / Low)
- 🤖 Multi-Agent system for analysis & recommendations
- 🔎 RAG-based health knowledge retrieval (FAISS)
- 📊 Interactive dashboard using Streamlit

---

## 🏗️ Architecture

User Input → ML Model → Risk Engine → Multi-Agent System → RAG Insights → Dashboard
